# ObsidianMD
- Markdown based note taking apps, unlike other proprietary solutions all your notes are stored locally with no server components.

## Plugins
Some plugins I personally use
- Excalidraw [(Online Version)](https://excalidraw.com/)
    - Whiteboard
- Text Transform
    - Change casing to title case, sentence case, etc.
- Code Editor shortcuts
    - Implements  move/copy line up down commonly seen in IDEs
- Another Sticky Headings
    - Makes heading stick so you can easily see levels of headings
- Iconize
    - Add icons to folders
- Latex Suite and Quick Latex
    - To write latex more easily
- Mermaid Tools
    - To make graphs using mermaid syntax more easily
